# 🐍 Python Course – MAAB Academy

This repository contains materials and notes from the **Python course for selected talented students** organized by **MAAB Academy**.

The course covers Python fundamentals, data analysis, and web scraping, with a focus on real-world applications.

---

## 📚 Course Overview

This program is designed to build strong programming skills and practical experience using modern Python tools.

### 🔧 Technologies & Libraries
- Python  
- Pandas  
- BeautifulSoup4  
- Requests  
- JSON / CSV  

---

## 🗓️ Curriculum

### 📌 Week 1–2: Python Fundamentals
- Data types: strings, numbers, lists, dictionaries, tuples  
- Loops (`for`, `while`) and conditions (`if-else`)  
- Functions (`def`)  
- Error handling with `try/except`  
- File handling (read/write files)  

---

### 📌 Week 3–4: Data Manipulation with Pandas
- Reading data from CSV, Excel, JSON  
- DataFrames and Series  
- Filtering, sorting, indexing  
- Handling missing data: `dropna()`, `fillna()`  
- GroupBy, aggregations, pivot tables  
- Merging and joining datasets  

---

### 📌 Week 5: Web Scraping & APIs
- Using `requests` to call APIs  
- Parsing JSON data  
- Introduction to REST APIs  
- HTML parsing with BeautifulSoup4  
- Scraping tables, tags, and attributes  

---

## 🚀 Final Project
- Load a real-world dataset  
- Clean and preprocess data using Pandas  
- Perform basic analysis
