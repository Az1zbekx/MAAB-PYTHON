Python Course for Selected Talented Youth by MAAB Academy

Topics covered during the course:

Python (Basics, Pandas, BeautifulSoup4, APIs, JSON, CSV)

Week 1–2: Python Fundamentals

Data types: strings, numbers, lists, dictionaries, tuples
Loops (for, while), conditions (if-else), functions (def)
Error handling using try/except blocks
File I/O: reading from and writing to files

Week 3–4: Data Manipulation with Pandas

Reading data from CSV, Excel, and JSON files
Working with DataFrames and Series
Filtering, sorting, and indexing data
Handling missing data: dropna(), fillna()
GroupBy operations, pivot tables, and aggregations
Merging and joining datasets

Week 5: Web Scraping & APIs

Using the requests library to call APIs
Parsing JSON responses
Introduction to REST APIs
Using BeautifulSoup4 for HTML parsing
Scraping tables, tags, and attributes

Final Project

Load a real-world dataset and clean it using Pandas
